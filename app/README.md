# PUBG 플레이어 분석 프로젝트

## 1. 프로젝트 개요

이 프로젝트는 **PUBG(PlayerUnknown's Battlegrounds)** 플레이어 데이터를 분석하기 위한 웹 애플리케이션입니다. **머신러닝 모델**을 활용하여 의심스러운 사용자 행동을 감지하고 시각화하며, 잠재적 핵 사용자를 식별하는 데 도움을 줍니다.

이 애플리케이션은 **공식 PUBG API**를 통해 플레이어의 최근 경기 데이터를 가져오고, 이를 처리한 후 **사전 학습된 XGBoost 및 Random Forest 모델**에 입력하여 악의적 활동의 가능성을 예측합니다. 결과와 다양한 데이터 시각화는 사용자 친화적인 웹 인터페이스에 표시됩니다.

## 2. 주요 기능

- **실시간 데이터 수집**: Steam 및 Kakao 플랫폼에서 PUBG API를 통해 플레이어 경기 데이터를 실시간으로 수집합니다.
- **ML 기반 예측**: XGBoost와 Random Forest 모델을 사용하여 사용자를 '정상' 또는 '의심' 범주로 분류합니다.
- **상호작용형 웹 인터페이스**: Flask 기반 웹 UI를 통해 사용자가 플레이어 이름을 입력하고 분석을 시작할 수 있습니다.
- **풍부한 데이터 시각화**:
  - **사용자 상태 분포**: 예측된 사용자 상태의 분포를 보여주는 막대 차트
  - **예측 확률 추이**: 경기 생존 시간에 따른 예측 확률의 변화를 보여주는 선 차트
  - **플레이어 통계 분석**: 로컬 데이터셋(`pre_oversampling_data.csv`)에서 다음을 포함한 분석 차트를 생성합니다:
    - **레이더 차트**: 정상 사용자와 차단된 사용자 그룹의 평균 통계를 비교합니다.
    - **이상치 히트맵**: 개별 차단된 플레이어의 통계적으로 비정상적인 지표를 강조합니다.
    - **타임라인 산점도**: 생존 시간과 킬 수의 관계를 시각화하며, 사용자 유형별로 색상을 구분합니다.

## 4. 설정 및 설치

### 1단계: 저장소 복제
프로젝트 파일을 다운로드하거나 저장소를 복제합니다.

### 2단계: 의존성 설치
Python이 설치되어 있는지 확인하고, 다음 명령어로 모든 필수 패키지를 설치합니다:

pip install -r requirements.txt

text

### 3단계: 애플리케이션 실행
터미널에서 메인 애플리케이션 파일을 실행합니다:

python app1.py

text

## 5. 사용 방법

1. 애플리케이션이 실행 중일 때, 웹 브라우저를 열고 다음 주소로 이동합니다:
http://127.0.0.1:5000

text

2. 분석하려는 PUBG 플레이어 이름을 입력 필드에 입력합니다.

3. 플레이어가 사용하는 플랫폼에 해당하는 **"Steam"** 또는 **"Kakao"** 버튼을 클릭합니다.

4. 애플리케이션이 데이터를 가져오고 분석 결과와 시각화를 페이지에 표시합니다.
